Etiqueta para las definiciones:

Etiquetas en html: Una etiqueta HTML es un término rodeado por un signo de menor y otro de mayor. Por lo general, existe una segunda etiqueta que delimita el final. Algunos elementos no necesitan etiqueta de cierre.

Las etiquetas HTML se escriben siempre en minúsculas (norma W3C).
